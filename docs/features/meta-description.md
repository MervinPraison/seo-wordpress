# Meta Description Optimization

Automatically generate and manage meta descriptions for better search engine visibility.

## Overview

Meta descriptions appear in search engine results and influence click-through rates. Praison SEO optimizes these automatically.

## Features

### Automatic Generation

If no custom meta description is set, the plugin automatically generates one from:
- Post excerpt (if available)
- First paragraph of content
- Truncated to optimal length (155-160 characters)

### Custom Meta Descriptions

Override automatic generation with custom meta descriptions:

1. Edit any post or page
2. Scroll to the "Praison SEO" meta box
3. Enter your custom meta description
4. Character counter shows optimal length

## Best Practices

| Guideline | Recommendation |
|-----------|----------------|
| Length | 155-160 characters |
| Keywords | Include primary keyword |
| Call to action | Encourage clicks |
| Uniqueness | Different for each page |
