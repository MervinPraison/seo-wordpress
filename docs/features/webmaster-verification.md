# Webmaster Verification

Verify site ownership with major search engines.

## Overview

Webmaster verification proves you own your site to search engines, giving you access to powerful webmaster tools.

## Supported Platforms

### Google Search Console

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your property
3. Choose "HTML tag" verification
4. Copy the content value
5. Enter in **Settings → Praison SEO → Verification**

```html
<meta name="google-site-verification" content="your-verification-code">
```

### Bing Webmaster Tools

1. Go to [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. Add your site
3. Choose "Meta tag" verification
4. Copy the content value
5. Enter in plugin settings

```html
<meta name="msvalidate.01" content="your-verification-code">
```

### Alexa

1. Get your Alexa verification code
2. Enter in plugin settings

```html
<meta name="alexaVerifyID" content="your-verification-code">
```

## Configuration

1. Go to **Settings → Praison SEO**
2. Find the "Webmaster Verification" section
3. Enter your verification codes
4. Save settings
5. Return to webmaster tools to verify

## Benefits of Verification

| Platform | Benefits |
|----------|----------|
| Google | Search performance data, indexing control, sitemap submission |
| Bing | Bing search insights, URL submission, SEO reports |
| Alexa | Traffic rankings, competitive analysis |
