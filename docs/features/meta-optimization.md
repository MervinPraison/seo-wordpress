## How It Works

When editing a post or page, you'll see the **Praison SEO** meta box below the editor.

## Meta Box Fields

| Field | Purpose | Limit |
|-------|---------|-------|
| **Meta Title** | Search result title | 60 chars |
| **Meta Description** | Search result snippet | 160 chars |

## Character Counter

The plugin shows a live character counter to help you stay within limits.

## Automatic Fallbacks

If you don't set custom meta:

- **Title** ‚Üí Uses post title
- **Description** ‚Üí Uses excerpt or first paragraph

## Best Practices

!!! tip "Write for humans"
    Include your keyword naturally. Don't stuff keywords.

!!! tip "Be compelling"
    Write descriptions that make users want to click.
