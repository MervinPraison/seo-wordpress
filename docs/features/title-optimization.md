# Title Optimization

Optimize page titles for both SEO and user experience.

## Overview

Page titles are the most important on-page SEO factor. Praison SEO provides complete control over title tags.

## Features

### Title Templates

Configure title patterns for different content types:

- **Posts**: `%title% | %sitename%`
- **Pages**: `%title% | %sitename%`
- **Categories**: `%category% Archives | %sitename%`
- **Tags**: `%tag% Archives | %sitename%`

### Available Variables

| Variable | Description |
|----------|-------------|
| `%title%` | Post/page title |
| `%sitename%` | WordPress site name |
| `%category%` | Category name |
| `%tag%` | Tag name |
| `%author%` | Author display name |
| `%date%` | Publication date |

### Custom Titles

Override the title for any post/page:

1. Edit the post or page
2. Find the "Praison SEO" meta box
3. Enter your custom title
4. Title length counter helps optimize

### Home Page Title

Set a specific title for your home page:

1. Go to **Settings → Praison SEO**
2. Configure home page title
3. Save settings

## Best Practices

| Guideline | Recommendation |
|-----------|----------------|
| Length | 50-60 characters |
| Keywords | Primary keyword near start |
| Branding | Site name at end |
| Uniqueness | Unique for each page |

## Title Separator

Customize the separator between title parts:

- Default: `|`
- Options: `|`, `-`, `»`, `•`
